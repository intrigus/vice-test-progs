This directory contains some test/demo programs for the PET DWW
(Double-W) hi-res graphics board.

Run them like this:

    xpet -model 3032 -petdww 3d-holyhalter-1.prg

or set up the directory as disk drive 8:

    xpet -default -model 3032 -petdww +truedrive -drive8type 0 -device8 1 -fs8 $PWD

3d-holyhalter-1.prg	demos with some animated shapes
3d-holyhalter-2.prg
