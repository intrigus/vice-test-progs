 PRODUCT NAME : .................................................. FLINALE
 GROUP        : ...................................... Akronyme Analogiker
 COMPETITION  : ..................................................... None
 PARTY        : ................................ In honor of Revision 2014
 PLATFORM     : ........... NTSC VIC-20 + 32K RAM and PAL VIC-20 + 24K RAM
 RELEASE DATE : ............................................... 17/04/2014
  
 SHORT
  DESCRIPTION : Slideshows for two newly created VIC-20 graphic modes:
                
                Flinale NTSC: 88 x 400 true interlace with 8 x 1 color
                resolution and separate colors for background, border and
                auxiliary on each rasterline (IFLI88)

                Flinale PAL: 104 x 256 with 8 x 1 color resolution and
                separate colors for background, border and auxiliary on
                each rasterline (FLI104)

 SUPPORT:     : ..................................................... Mike

 SUCCESSFULLY TESTED ON:

 .......... NTSC VIC-20 + 32K RAM and PAL VIC-20 + 32K RAM + uIEC/SD drive

 NOTE:
 
 ...........Flinale NTSC will *NOT* run on VICE (up to 2.4 at least, since
 .......................... interlace mode is not supported (yet) by VICE)

 INSTRUCTIONS:

 Three .d64-files are included. flinale-pal.d64 is for the PAL-VIC while
 flinale-ntsc.d64 and flinale-nts2.d64 are for the NTSC-VIC.
 
 Also included are .ppm-converters in separate directories, explained more
 thorougly in the techinfo.txt.
