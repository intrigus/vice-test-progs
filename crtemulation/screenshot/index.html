The purpose of this page is to test/check the output of all renderers and check if their output is correct and consistant.
  • scale 2x not tested
  • for 80-columns chips, non vertical stretched output is not tested

C64 VICII (PAL)

  • the classic "Pepto Palette" is the reference, however VICE uses its builtin color generator to calculate the actual values based on various parameters on the fly.
  • VICE uses custom luma ramps for the various emulated VICII variants (none will match pepto exactly!).
  • the CRT emulation paramaters are tuned so when all values are in "neutral" position, the result will be close to (but not equal to!) the "Pepto colors"
  • however, due to popular demand, saturation and contrast was increased a bit by default (just like a lot of people did at their TVs/Monitors)
